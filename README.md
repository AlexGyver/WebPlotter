# WebPlotter
Строит график из данных из

- Serial (COM)
- WebSocket
- BLE

> [https://alexgyver.github.io/WebPlotter](https://alexgyver.github.io/WebPlotter)

## Формат пакетов
### Данные
Префикс - `$`, разделитель - `,`, в конце - перенос строки

- `$значение\n`
- `$значение1,значение2\n`

### Лог
Префикс - `@`, в конце - перенос строки

`$текст\n`

### Подписи
Префикс - `#`, разделитель - `,`, в конце - перенос строки

- `$подпись\n`
- `$подпись1,подпись2\n`

Указание единиц измерения в квадратных скобках в конце строки: `$подпись1[%],подпись2\n`.

Отображение пунктирной линией - знаки тире в начале строки: `$-подпись1,--подпись2\n`. Чем больше тире - тем длиннее штрихи, можно комбинировать с единицами измерения.
